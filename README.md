# Nail Salon Appointment Booking System 💅

## Description
Une application web pour permettre aux clients de prendre rendez-vous dans une onglerie. Système simple et efficace de gestion des réservations.

## Fonctionnalités
✅ Calendrier interactif
✅ Sélection des services (manucure, pédicure, décoration, etc.)
✅ Choix de l'horaire disponible
✅ Formulaire de client
✅ Confirmation de rendez-vous par email (simulation)
✅ Affichage des rendez-vous
✅ Responsive design (mobile-friendly)

## Installation

1. Clonez le repository:
```bash
git clone https://github.com/Nour2808/Onglerie-appointment.git
cd Onglerie-appointment
```

2. Ouvrez `index.html` dans votre navigateur ou utilisez un serveur local:
```bash
python -m http.server 8000
# ou
npx http-server
```

## Utilisation

1. Sélectionnez une date sur le calendrier
2. Choisissez votre service
3. Sélectionnez un horaire disponible
4. Remplissez votre formulaire
5. Confirmez votre rendez-vous
